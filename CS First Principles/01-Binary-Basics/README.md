# Project 01: Binary Basics

## Overview

This project teaches you how computers represent ALL data as binary (0s and 1s) from first principles. You'll understand not just HOW to convert between formats, but WHY computers work this way at the hardware level.

## What You'll Learn

- **The fundamental reason** computers use binary (hint: it's about transistors!)
- How to convert between decimal, binary, and hexadecimal
- How negative numbers work (two's complement)
- How decimals work (IEEE 754 floating-point)
- How text is encoded (ASCII and UTF-8)
- The trade-offs between precision, range, and storage

## Files

- **GUIDE.md** - Interactive educational guide with questions and explanations
- **binary_converter.py** - Implementation challenges
